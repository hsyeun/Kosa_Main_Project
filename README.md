팀원2 - 데이터 수집 & 전처리, 자소서 모델 구현, 백엔드 지원

  * 유저 맞춤 기업 추천 서비스

  * 챗봇
    - 취업 관련 상담 서비스  
    - 기업 관련 문의 서비스
